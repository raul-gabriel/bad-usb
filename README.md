# BadUSB Educativo con Arduino

Proyecto educativo para demostrar cómo un dispositivo USB puede emular un teclado y ejecutar comandos automáticamente.  
**Creado con fines educativos para feria universitaria.**

---

## 🎯 Objetivo

Al conectar el Arduino, este actúa como un teclado USB y abre automáticamente un video de YouTube (o cualquier enlace).

---

## Placas Compatibles

Este código funciona con placas que tengan **ATMega32U4** o soporte nativo de USB HID:

- **Arduino Leonardo**
- **Arduino Micro**
- **SparkFun Pro Micro** (ATMega32U4)
- **Arduino Leonardo ETH**
- **Teensy 2.0** (parcialmente)
- **Cualquier clon chino con ATMega32U4**

**No funciona** con Arduino Uno, Nano, Mega (no tienen USB HID nativo).

---

## Requisitos

- Arduino IDE (recomendado versión 2.x)
- Placa con **ATMega32U4**
- Cable USB
- Computadora objetivo con **Windows** (probado en Win 10 y 11)
- Conocimientos básicos de Arduino
